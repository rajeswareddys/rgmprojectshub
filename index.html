Description of our project

current days all things are going online. The proposed project will help the students to store the project information like source code and documentation. It helps to the upcoming students where students have ability to refer the old projects and to make changes on it and it provides better user interface where user can easily analyse each project with irrespective of language and it provide proper guidance .

In olden days people preferred to store books in library. Sometimes we may lost the books which contains useful information. This project helps to overcome this problem by using cloud concept where user can store data directly in the cloud where there is a less chance to loss the data.

The proposed project enables the students to upload their projects in zip format which reduces loading burden. It provides the facility to download documents or files from anywhere with the help of internet and a user can access data.
